Fetch on remote server where your pipeline is working

Option 1: Using git checkout for a single file

From the repo folder on remote server:

cd /path/to/pipeline2

git fetch origin main
git checkout origin/main -- app.html


Explanation:

git fetch origin main → fetches the latest changes from GitHub without merging.

git checkout origin/main -- index.html → updates only index.html to the latest version from GitHub.
